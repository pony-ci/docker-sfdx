# docker-sfdx

[![docker hub](https://img.shields.io/docker/pulls/ponyci/sfdx.svg)](https://hub.docker.com/repository/docker/ponyci/sfdx)

Run sfdx commands in docker.

### What's in this image
* ubuntu
* sfdx-cli
* jq
* git

### Verify version
`docker run --rm ponyci/sfdx sfdx version`
